TokoVOIP
TokoVOIP ist unser spielinterner VOIP-Ersatz mit einem maßgeschneiderten TS-Plugin.

Es gibt uns viel mehr Möglichkeiten, die derzeit solche Dinge erlauben, wie:
  • Bessere Sprachqualität
  • Funk System
  • Der Proximity-Chat ist für jeden Spieler einzigartig
  • Telefon System
  • Funk klickt Sound
  • Funk Effect

TokoVOIP muß sein Eingerichtet und Laufen auf unseren Servern zu spielen.

Wenn Sie dem entsprechenden Serverkanal nicht beitreten können, treten Sie bitte bei Channel in Teamspeak, so dass ein Support-Teammitglied Ihnen helfen kann.

Mehr Information und Download Links finden sie auf: https://elysium-rp.de/download/tokovoip_1_2_5.ts3_plugin
